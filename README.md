# 🚀 PlaceMate AI

An AI-powered Job & Internship Portal built with the MERN Stack that helps users discover jobs, analyze resume-job compatibility using AI, and prepare for interviews with AI-generated questions.

## ✨ Features

- 🔐 JWT Authentication
- 👤 User & Admin Roles
- 💼 Job & Company Management
- 📄 Resume Upload
- 🤖 AI Resume Match Analysis
- 💬 AI Interview Question Generation
- 📌 Saved Jobs
- 📱 Responsive UI

## 🛠️ Tech Stack

- **Frontend:** React, Vite, Tailwind CSS
- **Backend:** Node.js, Express.js, MongoDB
- **AI:** Groq (Llama 3.3 70B)
- **Other:** JWT, Cloudinary, Brevo

## 🚀 Getting Started

```bash
# Clone the repository
git clone <repository-url>

# Install dependencies
cd server && npm install
cd ../client && npm install
cd ../admin && npm install
```

Create a `.env` file inside the `server` directory and add the required environment variables.
